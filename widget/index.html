  API pública — úsala directamente
  
Si prefieres construir tu propia integración, puedes consultar la API de mindicador.cl directamente. Es gratuita, no requiere API key y tiene CORS habilitado.


  

    GET
    https://mindicador.cl/api
    Todos los indicadores del día (UF, dólar, UTM, euro, IPC…)
  

  

    GET
    https://mindicador.cl/api/uf
    UF del mes actual — array de valores diarios
  

  

    GET
    https://mindicador.cl/api/uf/2026
    UF de todo el año 2026
  

  

    GET
    https://mindicador.cl/api/dolar
    Dólar observado del mes
  

  

    GET
    https://mindicador.cl/api/utm
    UTM del mes actual
  


  
  Ejemplo en JavaScript puro
  

    

      JavaScript — fetch a la API
      
    

    
// Obtener valor UF de hoy
fetch('https://mindicador.cl/api/uf')
  .then(r => r.json())
  .then(data => {
    const hoy = data.serie[0];
    console.log(`UF hoy: $${hoy.valor} (${hoy.fecha})`);
    // → UF hoy: $38542.45 (2026-04-21T00:00:00.000Z)
  });

// Obtener todos los indicadores del día
fetch('https://mindicador.cl/api')
  .then(r => r.json())
  .then(data => {
    console.log('UF:', data.uf.valor);
    console.log('Dólar:', data.dolar.valor);
    console.log('UTM:', data.utm.valor);
  });
